Feature Frontend Chat Widget

This page demonstrates how to embed the Feature Frontend chatbot widget in a standalone site. The script tag below loads widget.js from your FastAPI deployment and provides a data-embed-token that authenticates the chat session. 

<script
  src="http://localhost:8000/widget.js"
  data-embed-token="demo-token"
  async></script>

Try clicking the floating chat bubble in the bottom-right corner to open the sandboxed iframe. The iframe points to /embed/chat on the same origin as the script file.
